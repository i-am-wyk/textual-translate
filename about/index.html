Textual是什么¶

Textual 是 python 的一个 TUI 库

TUI是什么

TUI就是文本用户界面(Text User Interface)，你可以理解为终端的UI

“TUI是指文本用户界面(Text-based User Interface)，通过文本实现交互窗口展示内容，定位光标和鼠标实现用户交互。” ——百度百科
