Getting started

开始构建 Textual应用程序 所需的一切(就是搭建开发/运行环境)。

要求¶

Textual 需要 Python3.7 或更高版本 (如果你有别的选择，请使用最新的 Python 版本)。Textual 可以在 Linux、macOS、Windows 以及其他所有能运行 Python 的操作系统上运行。

你的平台

Linux (所有发行版)¶

所有的 Linux 发行版都有一个可以运行 Textual 应用程序的终端模拟器。
