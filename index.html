Textual是什么？¶

Textual 是一个用于构建在终端中运行的应用程序的框架。文本用户界面(TUIs)与 web 和桌面应用程序相比有许多优势。

  • 快速开发

    使用你现有的 Python 技能构建出漂亮的用户界面。

  • 系统要求低

    系统要求低。如果你需要，可以在单板机上运行 Textual 。

  • 跨平台

    Textual 可以在任何地方运行。

  • 远程

    Textual 应用程序可以在 SSH 上运行。

  • CLI 集成

    Textual 应用程序可以从命令提示符启动和运行。

  • MIT开源协议

    Textual 基于 MIT 协议开源。
